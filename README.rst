rivt-codespace-examples
------------------------

#. click the *Fork* button in the upper right corner of the page to create a copy of the repository in your GitHub account
#. change to your GitHub account and open the forked repository
#. click the green *Code* button on the GitHub page 
#. select the *Codespaces* tab
#. click the *rivt-codespace-examples* option to create and open a *rivt Codespace* with the examples
